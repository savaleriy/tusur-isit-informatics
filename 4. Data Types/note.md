### ^2. **O(log n) ‚Äî –ª–æ–≥–∞—Ä–∏—Ñ–º–∏—á–µ—Å–∫–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // ^2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // ^2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(n - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
```

### 6. **O(n!) ‚Äî —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª—å–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å**

```python
from itertools import permutations

def brute_force_tsp(cities):
    best_route = None
    best_distance = float('inf')
    for route in permutations(cities):
        dist = calculate_total_distance(route)
        if dist < best_distance:
            best_distance = dist
            best_route = route
    return best_route

def calculate_total_distance(route):
    # –ü—Ä–æ—Å—Ç–æ —Å—É–º–º–∞ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π –º–µ–∂–¥—É —Å–æ—Å–µ–¥–Ω–∏–º–∏ –≥–æ—Ä–æ–¥–∞–º–∏
    return sum(abs(route[i] - route[i+1]) for i in range(len(route)-1))
```

–°—Ç–µ–∫–∏ –Ω–∞—Ö–æ–¥—è—Ç —à–∏—Ä–æ–∫–æ–µ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–µ –≤ –∞–ª–≥–æ—Ä–∏—Ç–º–∞—Ö –∏ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏. –ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –≤–∫–ª—é—á–∞—é—Ç —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é —Ñ—É–Ω–∫—Ü–∏–∏ "–û—Ç–º–µ–Ω–∏—Ç—å –≤–≤–æ–¥" –≤ —Ç–µ–∫—Å—Ç–æ–≤—ã—Ö —Ä–µ–¥–∞–∫—Ç–æ—Ä–∞—Ö, –≥–¥–µ –∫–∞–∂–¥–∞—è –ø—Ä–∞–≤–∫–∞ –ø–æ–º–µ—â–∞–µ—Ç—Å—è –≤ —Å—Ç–µ–∫, –∞ –∑–∞—Ç–µ–º –º–æ–∂–µ—Ç –±—ã—Ç—å –æ—Ç–º–µ–Ω–µ–Ω–∞ –ø—É—Ç–µ–º –∏–∑–≤–ª–µ—á–µ–Ω–∏—è –∏–∑ —Å—Ç–µ–∫–∞.

**–î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:**

1. **–ü—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞–ª–∏—á–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ —Å—Ç–µ–∫–µ (`isEmpty()`):** –ü–æ–∑–≤–æ–ª—è–µ—Ç —É–∑–Ω–∞—Ç—å, –ø—É—Å—Ç –ª–∏ —Å—Ç–µ–∫.

–°—Ç–µ–∫–∏ –æ–±–µ—Å–ø–µ—á–∏–≤–∞—é—Ç –ø—Ä–æ—Å—Ç–æ—Ç—É –∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã—Ö –∑–∞–¥–∞—á, –≥–¥–µ –≤–∞–∂–µ–Ω –ø–æ—Ä—è–¥–æ–∫ –æ–±—Ä–∞–±–æ—Ç–∫–∏ –¥–∞–Ω–Ω—ã—Ö.

### –û—á–µ—Ä–µ–¥—å —Å¬†–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º

–û—á–µ—Ä–µ–¥—å —Å¬†–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º (**priority queue**) –∞–Ω–∞–ª–æ–≥–∏—á–Ω–∞ –æ–±—ã—á–Ω–æ–π –æ—á–µ—Ä–µ–¥–∏ —Å —Ç–æ–π –ª–∏—à—å —Ä–∞–∑–Ω–∏—Ü–µ–π, —á—Ç–æ –ø–æ–º–µ—â–µ–Ω–Ω—ã–º –≤ –Ω–µ–µ —ç–ª–µ–º–µ–Ω—Ç–∞–º –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è **–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç**. –õ—é–¥–∏, –æ–∂–∏–¥–∞—é—â–∏–µ –º–µ–¥–∏—Ü–∏–Ω—Å–∫–æ–π –ø–æ–º–æ—â–∏ –≤ –±–æ–ª—å–Ω–∏—Ü–µ, ‚Äî –≤–æ—Ç —Ä–µ–∞–ª—å–Ω—ã–π –ø—Ä–∏–º–µ—Ä –æ—á–µ—Ä–µ–¥–∏ —Å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º. –≠–∫—Å—Ç—Ä–µ–Ω–Ω—ã–µ —Å–ª—É—á–∞–∏ –ø–æ–ª—É—á–∞—é—Ç –≤—ã—Å—à–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –∏ –ø–µ—Ä–µ—Ö–æ–¥—è—Ç –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ –≤ –Ω–∞—á–∞–ª–æ –æ—á–µ—Ä–µ–¥–∏, —Ç–æ–≥–¥–∞ –∫–∞–∫ –Ω–µ–∑–Ω–∞—á–∏—Ç–µ–ª—å–Ω—ã–µ –¥–æ–±–∞–≤–ª—è—é—Ç—Å—è –≤ –µ–µ –∫–æ–Ω–µ—Ü. –û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏, —Ä–µ–∞–ª–∏–∑—É–µ–º—ã–µ –æ—á–µ—Ä–µ–¥—å—é —Å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º, —Ç–∞–∫–æ–≤—ã
- `enqueue(e, p)` ‚Äî –¥–æ–±–∞–≤–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç $e$ –≤ –æ—á–µ—Ä–µ–¥—å —Å–æ–≥–ª–∞—Å–Ω–æ —É—Ä–æ–≤–Ω—é **–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–æ—Å—Ç–∏** $p$; 
- `dequeue()` ‚Äî –≤–µ—Ä–Ω—É—Ç—å —ç–ª–µ–º–µ–Ω—Ç, —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–Ω—ã–π –≤ –Ω–∞—á–∞–ª–µ –æ—á–µ—Ä–µ–¥–∏, –∏ —É–¥–∞–ª–∏—Ç—å –µ–≥–æ.

–í –∫–æ–º–ø—å—é—Ç–µ—Ä–µ, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –º–Ω–æ–≥–æ —Ä–∞–±–æ—á–∏—Ö –ø—Ä–æ—Ü–µ—Å—Å–æ–≤ ‚Äî –∏ –≤—Å–µ–≥–æ –æ–¥–∏–Ω –∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ –¶–ü, –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–Ω—ã—Ö –¥–ª—è –∏—Ö –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è. –û–ø–µ—Ä–∞—Ü–∏–æ–Ω–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ —Å—Ç–∞–≤–∏—Ç –≤—Å–µ –ø—Ä–æ—Ü–µ—Å—Å—ã, –æ–∂–∏–¥–∞—é—â–∏–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è, –≤ –æ—á–µ—Ä–µ–¥—å —Å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º. –ö–∞–∂–¥—ã–π –ø—Ä–æ—Ü–µ—Å—Å –ø–æ–ª—É—á–∞–µ—Ç —Å–≤–æ–π —É—Ä–æ–≤–µ–Ω—å **–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–æ—Å—Ç–∏**. –û–ø–µ—Ä–∞—Ü–∏–æ–Ω–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ –∏—Å–∫–ª—é—á–∞–µ—Ç –ø—Ä–æ—Ü–µ—Å—Å –∏–∑ –æ—á–µ—Ä–µ–¥–∏ –∏ –ø–æ–∑–≤–æ–ª—è–µ—Ç –µ–º—É –Ω–µ–∫–æ—Ç–æ—Ä–æ–µ –≤—Ä–µ–º—è –ø–æ—Ä–∞–±–æ—Ç–∞—Ç—å. –ü–æ–∑–¥–Ω–µ–µ, –µ—Å–ª–∏ –ø—Ä–æ—Ü–µ—Å—Å –Ω–µ –∑–∞–≤–µ—Ä—à–∏–ª—Å—è, –æ–Ω —Å–Ω–æ–≤–∞ —Å—Ç–∞–≤–∏—Ç—Å—è –≤ –æ—á–µ—Ä–µ–¥—å. –û–ø–µ—Ä–∞—Ü–∏–æ–Ω–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ —Ä–∞–∑ –∑–∞ —Ä–∞–∑–æ–º –ø–æ–≤—Ç–æ—Ä—è–µ—Ç —ç—Ç—É –ø—Ä–æ—Ü–µ–¥—É—Ä—É. –ù–µ–∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–æ—Ü–µ—Å—Å—ã –±–æ–ª–µ–µ –≤–∞–∂–Ω—ã –∏ –±–µ–∑–æ—Ç–ª–∞–≥–∞—Ç–µ–ª—å–Ω–æ –ø–æ–ª—É—á–∞—é—Ç –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–Ω–æ–µ –≤—Ä–µ–º—è, –¥—Ä—É–≥–∏–µ –æ–∂–∏–¥–∞—é—Ç –≤ –æ—á–µ—Ä–µ–¥–∏ –¥–æ–ª—å—à–µ. –ü—Ä–æ—Ü–µ—Å—Å, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–ª—É—á–∞–µ—Ç –≤–≤–æ–¥ —Å –∫–ª–∞–≤–∏–∞—Ç—É—Ä—ã, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –ø–æ–ª—É—á–∞–µ—Ç —Å–∞–º—ã–π –≤—ã—Å–æ–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç ‚Äî –≤–µ–¥—å –µ—Å–ª–∏ –∫–æ–º–ø—å—é—Ç–µ—Ä –Ω–µ —Ä–µ–∞–≥–∏—Ä—É–µ—Ç –Ω–∞ –Ω–∞–∂–∞—Ç–∏—è –∫–ª–∞–≤–∏—à, —Ç–æ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –º–æ–∂–µ—Ç –ø–æ–¥—É–º–∞—Ç—å, —á—Ç–æ –æ–Ω –∑–∞–≤–∏—Å, –∏ –ø–æ–ø—Ä–æ–±—É–µ—Ç —Å–¥–µ–ª–∞—Ç—å ¬´—Ö–æ–ª–æ–¥–Ω—ã–π¬ª –ø–µ—Ä–µ–∑–∞–ø—É—Å–∫, —á—Ç–æ –≤—Å–µ–≥–¥–∞ –≤—Ä–µ–¥–Ω–æ.

### –°–ø–∏—Å–æ–∫

–ü—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å –≥—Ä—É–ø–ø–∞–º–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏–Ω–æ–≥–¥–∞ —Ç—Ä–µ–±—É–µ—Ç—Å—è –≥–∏–±–∫–æ—Å—Ç—å. –ù–∞–ø—Ä–∏–º–µ—Ä, –º–æ–∂–µ—Ç –≤–æ–∑–Ω–∏–∫–Ω—É—Ç—å –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å –≤ –ø–µ—Ä–µ—É–ø–æ—Ä—è–¥–æ—á–∏–≤–∞–Ω–∏–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏–ª–∏ –∏—Ö –∏–∑–≤–ª–µ—á–µ–Ω–∏–∏, –≤—Å—Ç–∞–≤–∫–µ –∏ —É–¥–∞–ª–µ–Ω–∏–∏ –≤ –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ. –í —Ç–∞–∫–∏—Ö —Å–ª—É—á–∞—è—Ö —É–¥–æ–±–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–ø–∏—Å–æ–∫ (`list`). –ß–∞—â–µ –≤—Å–µ–≥–æ **–ê–¢–î** `–°–ø–∏—Å–æ–∫` –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:

- `insert(n, e)` ‚Äî –≤—Å—Ç–∞–≤–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç `e` –≤ –ø–æ–∑–∏—Ü–∏—é `n`;
- `remove(n)` ‚Äî —É–¥–∞–ª–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç, –Ω–∞—Ö–æ–¥—è—â–∏–π—Å—è –≤ –ø–æ–∑–∏—Ü–∏–∏ `n`;
- `get(n)` ‚Äî –ø–æ–ª—É—á–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç, –Ω–∞—Ö–æ–¥—è—â–∏–π—Å—è –≤ –ø–æ–∑–∏—Ü–∏–∏ `n`;
- `sort()` ‚Äî –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã;
- `slice(start, end)` ‚Äî –≤–µ—Ä–Ω—É—Ç—å —Ñ—Ä–∞–≥–º–µ–Ω—Ç —Å–ø–∏—Å–∫–∞, –Ω–∞—á–∏–Ω–∞—é—â–∏–π—Å—è —Å –ø–æ–∑–∏—Ü–∏–∏ `start` –∏ –∑–∞–∫–∞–Ω—á–∏–≤–∞—é—â–∏–π—Å—è –≤ –ø–æ–∑–∏—Ü–∏–∏ `end`;
- `reverse()` ‚Äî –∏–∑–º–µ–Ω–∏—Ç—å –ø–æ—Ä—è–¥–æ–∫ —Å–ª–µ–¥–æ–≤–∞–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –Ω–∞ –æ–±—Ä–∞—Ç–Ω—ã–π.

–°–ø–∏—Å–æ–∫ —è–≤–ª—è–µ—Ç—Å—è –æ–¥–Ω–∏–º –∏–∑ –Ω–∞–∏–±–æ–ª–µ–µ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã—Ö **–ê–¢–î**. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –≤–∞–º –Ω—É–∂–Ω–æ —Ö—Ä–∞–Ω–∏—Ç—å —Å—Å—ã–ª–∫–∏ –Ω–∞ —á–∞—Å—Ç–æ –∑–∞–ø—Ä–∞—à–∏–≤–∞–µ–º—ã–µ —Ñ–∞–π–ª—ã –≤ —Å–∏—Å—Ç–µ–º–µ, —Ç–æ —Å–ø–∏—Å–æ–∫ ‚Äî –∏–¥–µ–∞–ª—å–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ: –≤—ã –º–æ–∂–µ—Ç–µ —Å–æ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å —Å—Å—ã–ª–∫–∏ –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è –∏ —É–¥–∞–ª—è—Ç—å –∏—Ö, –µ—Å–ª–∏ –∫ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–º —Ñ–∞–π–ª–∞–º –æ–±—Ä–∞—â–∞—é—Ç—Å—è —Ä–µ–∂–µ.

–°—Ç–µ–∫ –∏ –æ—á–µ—Ä–µ–¥–∏ —Å–ª–µ–¥—É–µ—Ç –ø—Ä–µ–¥–ø–æ—á–∏—Ç–∞—Ç—å, –∫–æ–≥–¥–∞ –≥–∏–±–∫–æ—Å—Ç—å, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ–º–∞—è —Å–ø–∏—Å–∫–æ–º, –Ω–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è. –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –±–æ–ª–µ–µ –ø—Ä–æ—Å—Ç–æ–≥–æ **–ê–¢–î** –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç, —á—Ç–æ –¥–∞–Ω–Ω—ã–µ –±—É–¥—É—Ç –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å—Å—è —Å—Ç—Ä–æ–≥–∏–º –∏ –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã–º –æ–±—Ä–∞–∑–æ–º (–ø–æ –ø—Ä–∏–Ω—Ü–∏–ø—É **FIFO** –∏–ª–∏ **LIFO**). –≠—Ç–æ —Ç–∞–∫–∂–µ –¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ —è—Å–Ω—ã–º: –∑–Ω–∞—è, —á—Ç–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å—Ç–µ–∫, –ª–µ–≥—á–µ –ø–æ–Ω—è—Ç—å —Ö–∞—Ä–∞–∫—Ç–µ—Ä –ø–æ—Ç–æ–∫–æ–≤ –¥–∞–Ω–Ω—ã—Ö –Ω–∞ –≤—Ö–æ–¥–µ –∏ –≤—ã—Ö–æ–¥–µ.

### –°–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫

–°–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ (sorted list) –æ–∫–∞–∑—ã–≤–∞–µ—Ç—Å—è –ø–æ–ª–µ–∑–Ω—ã–º, –∫–æ–≥–¥–∞ —Ç—Ä–µ–±—É–µ—Ç—Å—è –ø–æ—Å—Ç–æ—è–Ω–Ω–∞—è —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ—Å—Ç—å —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –í —Ç–∞–∫–∏—Ö —Å–ª—É—á–∞—è—Ö, –≤–º–µ—Å—Ç–æ –ø–æ–∏—Å–∫–∞ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–π –ø–æ–∑–∏—Ü–∏–∏ –ø–µ—Ä–µ–¥ –∫–∞–∂–¥–æ–π –≤—Å—Ç–∞–≤–∫–æ–π –≤ —Å–ø–∏—Å–æ–∫ (–∏ –ø–µ—Ä–∏–æ–¥–∏—á–µ—Å–∫–æ–π —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏ –µ–≥–æ –≤—Ä—É—á–Ω—É—é), –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫. –ù–µ–≤–∞–∂–Ω–æ, —á—Ç–æ –º—ã –ø–æ–º–µ—â–∞–µ–º –≤ –Ω–µ–≥–æ, —ç–ª–µ–º–µ–Ω—Ç—ã –≤—Å–µ–≥–¥–∞ –±—É–¥—É—Ç —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω—ã –ø–æ –ø–æ—Ä—è–¥–∫—É. –ù–∏ –æ–¥–Ω–∞ –∏–∑ –æ–ø–µ—Ä–∞—Ü–∏–π —ç—Ç–æ–≥–æ **–ê–¢–î** –Ω–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç –∏–∑–º–µ–Ω—è—Ç—å –ø–æ—Ä—è–¥–æ–∫ –µ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –°–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –º–µ–Ω—å—à–µ –æ–ø–µ—Ä–∞—Ü–∏–π –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å –æ–±—ã—á–Ω—ã–º —Å–ø–∏—Å–∫–æ–º:

- `insert(e)` ‚Äî –≤—Å—Ç–∞–≤–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç `e` –≤ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—É—é –ø–æ–∑–∏—Ü–∏—é –≤ —Å–ø–∏—Å–∫–µ;

- `get(n)` ‚Äî –ø–æ–ª—É—á–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç, –Ω–∞—Ö–æ–¥—è—â–∏–π—Å—è –≤ –ø–æ–∑–∏—Ü–∏–∏ `n`.

## –°–ª–æ–≤–∞—Ä—å

–°–ª–æ–≤–∞—Ä—å –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–π –º–µ–∂–¥—É –¥–≤—É–º—è –æ–±—ä–µ–∫—Ç–∞–º–∏: –∫–ª—é—á–æ–º (`key`) –∏ –∑–Ω–∞—á–µ–Ω–∏–µ–º (`value`). –í—ã –º–æ–∂–µ—Ç–µ –æ—Å—É—â–µ—Å—Ç–≤–∏—Ç—å –ø–æ–∏—Å–∫ –ø–æ –∫–ª—é—á—É –∏ –ø–æ–ª—É—á–∏—Ç—å —Å–≤—è–∑–∞–Ω–Ω–æ–µ —Å –Ω–∏–º –∑–Ω–∞—á–µ–Ω–∏–µ. –°–ª–æ–≤–∞—Ä—å —Ö–æ—Ä–æ—à–æ –ø–æ–¥—Ö–æ–¥–∏—Ç, –Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏–æ–Ω–Ω—ã—Ö –Ω–æ–º–µ—Ä–æ–≤ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π –≤ –∫–∞—á–µ—Å—Ç–≤–µ –∫–ª—é—á–µ–π –∏ –ø–æ–ª–Ω—ã—Ö –∏–º–µ–Ω –≤ –∫–∞—á–µ—Å—Ç–≤–µ –∑–Ω–∞—á–µ–Ω–∏–π. –¢–∞–∫–æ–π —Å–ª–æ–≤–∞—Ä—å –ø–æ –∑–∞–¥–∞–Ω–Ω–æ–º—É –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏–æ–Ω–Ω–æ–º—É –Ω–æ–º–µ—Ä—É –≤–µ—Ä–Ω–µ—Ç —Å–≤—è–∑–∞–Ω–Ω–æ–µ —Å –Ω–∏–º –∏–º—è. –°—É—â–µ—Å—Ç–≤—É—é—Ç —Å–ª–µ–¥—É—é—â–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –¥–ª—è —Å–ª–æ–≤–∞—Ä–µ–π:

- `set(key, value)` ‚Äî –¥–æ–±–∞–≤–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç —Å –∑–∞–¥–∞–Ω–Ω—ã–º –∫–ª—é—á–æ–º –∏ –∑–Ω–∞—á–µ–Ω–∏–µ–º;

- `get(key)` ‚Äî –ø–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ, —Å–≤—è–∑–∞–Ω–Ω–æ–µ —Å –∫–ª—é—á–æ–º `key`.

### –ú–Ω–æ–∂–µ—Å—Ç–≤–æ

–ú–Ω–æ–∂–µ—Å—Ç–≤–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã–µ –≥—Ä—É–ø–ø—ã —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã–µ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–º –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º. –≠—Ç–æ—Ç **–ê–¢–î** –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è, –∫–æ–≥–¥–∞ –ø–æ—Ä—è–¥–æ–∫ —Å–ª–µ–¥–æ–≤–∞–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –Ω–µ –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏—è, –ª–∏–±–æ –∫–æ–≥–¥–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –æ–±–µ—Å–ø–µ—á–∏—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω–æ—Å—Ç—å —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –≥—Ä—É–ø–ø–µ. –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –Ω–∞–±–æ—Ä –æ–ø–µ—Ä–∞—Ü–∏–π –¥–ª—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –≤–∫–ª—é—á–∞–µ—Ç –≤ —Å–µ–±—è:

- `add(e)` ‚Äî –¥–æ–±–∞–≤–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏–ª–∏ –≤–µ—Ä–Ω—É—Ç—å –æ—à–∏–±–∫—É, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç —É–∂–µ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ;
- `list()` ‚Äî –ø–µ—Ä–µ—á–∏—Å–ª–∏—Ç—å –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã, –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É—é—â–∏–µ –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ;
- `delete(e)` ‚Äî —É–¥–∞–ª–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–∞.

**–ê–¢–î** –¥–ª—è –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–∞ –ø–æ–¥–æ–±–µ–Ω –ø—Ä–∏–±–æ—Ä–Ω–æ–π –ø–∞–Ω–µ–ª–∏ –¥–ª—è –≤–æ–¥–∏—Ç–µ–ª—è. –û–¥–Ω–∞–∫–æ –¥–∞–≤–∞–π—Ç–µ –≤—Å–µ-—Ç–∞–∫–∏ –ø–æ–ø—Ä–æ–±—É–µ–º —Ä–∞–∑–æ–±—Ä–∞—Ç—å—Å—è, –∫–∞–∫ –ø—Ä–æ–≤–æ–¥–∞ –ø—Ä–æ–ª–æ–∂–µ–Ω—ã –∑–∞ —ç—Ç–æ–π –ø—Ä–∏–±–æ—Ä–Ω–æ–π –ø–∞–Ω–µ–ª—å—é.

## –°—Ç—Ä—É–∫—Ç—É—Ä—ã

**–ê–¢–î** –æ–ø–∏—Å—ã–≤–∞–µ—Ç —Ç–æ–ª—å–∫–æ, *–∫–∞–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –º–æ–∂–Ω–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å —Å –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ —Ç–∏–ø–∞, –æ–ø—Ä–µ–¥–µ–ª—è—è —Å–ø–∏—Å–æ–∫ –æ–ø–µ—Ä–∞—Ü–∏–π, –Ω–æ –Ω–µ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—è –¥–µ—Ç–∞–ª–µ–π –æ —Ç–æ–º, –∫–∞–∫ –æ–Ω–∏ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω—ã*. **–°—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö**, –Ω–∞–ø—Ä–æ—Ç–∏–≤, *–æ–ø–∏—Å—ã–≤–∞—é—Ç –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏—é –¥–∞–Ω–Ω—ã—Ö –∏ –¥–æ—Å—Ç—É–ø –∫ –Ω–∏–º –≤ –ø–∞–º—è—Ç–∏ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞*. 

>–û–Ω–∏ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—Ç —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω—ã—Ö —Ç–∏–ø–æ–≤ –¥–∞–Ω–Ω—ã—Ö –≤ –º–æ–¥—É–ª—è—Ö –æ–±—Ä–∞–±–æ—Ç–∫–∏ –¥–∞–Ω–Ω—ã—Ö.

–°—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Å–ø–æ—Å–æ–±–æ–≤ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ **–∞–±—Å—Ç—Ä–∞–∫—Ç–Ω—ã—Ö —Ç–∏–ø–æ–≤ –¥–∞–Ω–Ω—ã—Ö**, —Ç–∞–∫ –∫–∞–∫ —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Ä–∞–∑–Ω–æ–æ–±—Ä–∞–∑–Ω—ã–µ **—Å—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö**. –í—ã–±–æ—Ä –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω–æ–≥–æ —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä–∞—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Å—Ç—Ä—É–∫—Ç—É—Ä—É –¥–∞–Ω–Ω—ã—Ö, –Ω–∞–∏–ª—É—á—à–∏–º –æ–±—Ä–∞–∑–æ–º —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â—É—é –≤–∞—à–∏–º –ø–æ—Ç—Ä–µ–±–Ω–æ—Å—Ç—è–º, –∏–º–µ–µ—Ç –≤–∞–∂–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã—Ö –∫–æ–º–ø—å—é—Ç–µ—Ä–Ω—ã—Ö –ø—Ä–æ–≥—Ä–∞–º–º. –í –¥–∞–ª—å–Ω–µ–π—à–µ–º –º—ã —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –Ω–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã–µ —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö –∏ –∏–∑—É—á–∏–º –∏—Ö —Å–∏–ª—å–Ω—ã–µ –∏ —Å–ª–∞–±—ã–µ —Å—Ç–æ—Ä–æ–Ω—ã.

### –ú–∞—Å—Å–∏–≤

–ú–∞—Å—Å–∏–≤ (array) –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å–∞–º—ã–π –ø—Ä–æ—Å—Ç–æ–π —Å–ø–æ—Å–æ–± —Ö—Ä–∞–Ω–µ–Ω–∏—è –Ω–∞–±–æ—Ä–∞ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –ø–∞–º—è—Ç–∏ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞. –û–Ω –æ—Å–Ω–æ–≤–∞–Ω –Ω–∞ –≤—ã–¥–µ–ª–µ–Ω–∏–∏ –µ–¥–∏–Ω–æ–≥–æ –±–ª–æ–∫–∞ –ø–∞–º—è—Ç–∏ –∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ–π –∑–∞–ø–∏—Å–∏ –≤ –Ω–µ–≥–æ –≤–∞—à–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –ö–æ–Ω–µ—Ü –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –æ–±—ã—á–Ω–æ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º –º–∞—Ä–∫–µ—Ä–æ–º NULL.

–ö–∞–∂–¥—ã–π –æ–±—ä–µ–∫—Ç –≤ –º–∞—Å—Å–∏–≤–µ –∑–∞–Ω–∏–º–∞–µ—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π –æ–±—ä–µ–º –ø–∞–º—è—Ç–∏, –∞ –∏–º–µ–Ω–Ω–æ $b$ –±–∞–π—Ç. –ü—Ä–µ–¥—Å—Ç–∞–≤–∏–º –º–∞—Å—Å–∏–≤, –Ω–∞—á–∏–Ω–∞—é—â–∏–π—Å—è —Å –∞–¥—Ä–µ—Å–∞ —è—á–µ–π–∫–∏ –ø–∞–º—è—Ç–∏ $s$, –≥–¥–µ –∫–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç –∑–∞–Ω–∏–º–∞–µ—Ç $b$ –±–∞–π—Ç. –î–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è $n$-–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏–∑–≤–ª–µ—á—å $b$ –±–∞–π—Ç, –Ω–∞—á–∏–Ω–∞—è —Å –ø–æ–∑–∏—Ü–∏–∏ –≤ –ø–∞–º—è—Ç–∏ $s + (b √ó n)$. –≠—Ç–æ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –ø—Ä—è–º–æ–π –¥–æ—Å—Ç—É–ø –∫ –ª—é–±–æ–º—É —ç–ª–µ–º–µ–Ω—Ç—É –º–∞—Å—Å–∏–≤–∞.

```python
def find_node(binary_tree, value):
    node = binary_tree.root_node
    while node:
        if node.value == value:
            return node
        if value > node.value:

        else:
            node = node.left
    return "NOT FOUND"

```

```python
def insert_node(binary_tree, new_node):
    node = binary_tree.root_node
    while node:
        last_node = node
        if new_node.value > node.value:
            node = node.right
        else:
            node = node.left
    if new_node.value > last_node.value:
        last_node.right = new_node
    else:
        last_node.left = new_node
```
