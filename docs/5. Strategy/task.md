## 23. Найдите разделы продукта с максимальным или минимальным баллом 

Вам дано целое число $n$. Необходимо найти произведения его разбиений и соответствующие им специальные оценки.

**Определение**: Произведением разбиения числа $n$ называется набор множителей, которые в произведении дают $n$. Например, разбиение $[59,3,2,2,2]$ является разбиением числа $1416$, потому что:

$$59 \times 3 \times 2 \times 2 \times 2 = 1416$$

**Оценка**: Для каждого разбиения, состоящего из множителей $d_1, d_2, \ldots, d_k​$ с соответствующими частотами$f_1, f_2, \ldots, f_k$ оценка $sc$ вычисляется следующим образом:


$$sc = (d_1^{f_1} + d_2^{f_2} + \ldots + d_k^{f_k}) \times k$$


где $k$ --- количество уникальных множителей.

Напишите функцию `find_spec_prod_part(n, com)`, которая принимает два аргумента:

- целое число $n$ (где n>0n > 0n>0)
- строку $com$, которая может принимать значения `'max'` или `'min'`

Функция должна вернуть список из двух элементов: найденное разбиение произведения (в виде отсортированного по убыванию списка) и соответствующую оценку.

```python
find_spec_prod_part(1416, 'max')  # должно вернуть [[708, 2], 1420]
find_spec_prod_part(1416, 'min')  # должно вернуть [[59, 24], 166]
```
